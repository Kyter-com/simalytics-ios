# simalytics-ios

Native iOS companion for [Simkl](https://simkl.com) — track movies, TV shows, and anime in one place.

[Download on the App Store](https://apps.apple.com/us/app/simalytics/id6745519450)

## Stack

SwiftUI + SwiftData, iOS 17+. Observability via Sentry. Dependencies via SPM.

## Backend

Talks to [simalytics-api](https://github.com/Kyter-com/simalytics-api), a small Cloudflare Workers service handling Simkl OAuth exchange and a TMDB watch-providers proxy.
